# store

状态管理

```text
store/
├── app.ts                     //项目相关的一些基础信息
├── balances.ts                //资产相关
├── currency.ts                //币种相关
├── index.ts                   //聚合
├── market.ts                  //市场相关
├── trade.ts                   //交易相关
└── user.ts                    //用户相关
```

* 能不添加内容就不添加，除非你多次用到
* 添加内容一定要写明确的注释内容(注释格式参考现有规范)
